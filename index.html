Display: Lo que mostramos en nuestra pantalla

Display None

Este tipo de display es especial, porque nos permite dejar de visualizar contenido dentro de mi sitio.

Cuando ocupamos un display none, las etiquetas dejan de verse, pero sigue el código ahí.

Display Block

El elemento genera un cuadro del elemento de bloque.

Significa que mi bloque (recuerda que todo en HTML es un bloque), ocupa todo el espacio disponible de izquierda a derecha, o dicho de otra forma todo el renglón del componente padre.

Display InLine

El elemento genera uno o más cuadros de elemento en línea

Significa que el contenedor o bloque se adapta al ancho de su contenido hij@.

Display Inline-Block

Si tú le pones tamaño a un display block, funciona, pero sigue ocupando todo el renglón.

Si tú le pones tamaño a un display inline, ignora las propiedadesde alto (height) y ancho (width).

Esto puede ser problemático si yo quiero controlar el tamaño de mi componente, y que el contemnido esté en la misma línea, patra eso existe el display inline block.
